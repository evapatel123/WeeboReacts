# 🤖 Weebo Reacts

An interactive robot named Weebo built entirely with **HTML5 Canvas**, **CSS**, and **Vanilla JavaScript**.

Weebo follows your cursor with expressive eyes and head movements, reacts when clicked with animations and speech bubbles, gets annoyed if you keep poking it, and even dramatically disappears before returning.

Built as a fun demonstration of Canvas animation, UI interactions, and frontend creativity—without using any frameworks.

---

## ✨ Features

- Mouse tracking
- Dynamic facial expressions
- Random blinking animation
- Speech bubble reactions
- Particle effects
- Angry mode
- Collapse and reappear animation
- Floating idle animation
- Responsive design
- Pure HTML, CSS & JavaScript
- No external libraries
- HTML5 Canvas rendering

## 🎮 Interactions

### Mouse Movement

Weebo smoothly follows your cursor by moving:

- Head
- Eyes
- Body
- Logo

using interpolated animation for natural movement.

---

### Clicking

Clicking on Weebo triggers playful interactions.

- Speech bubbles
- Happy expressions
- Bounce animation
- Floating particles

---

### Angry Mode

After repeatedly clicking the robot, it becomes annoyed.

Its behavior changes with:

- Red glowing eyes
- Angry messages
- Sad facial expressions

---

### Collapse

Keep clicking...

Eventually Weebo gives up.

It dramatically disappears before returning a moment later.
